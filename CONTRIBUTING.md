# Contributing to Quantalytics

We love your input! We want to make contributing to Quantalytics as easy and transparent as
possible.

## Getting started

1. Fork the repo and create your branch from `main`.
2. Install dependencies with `pip install -e .[dev]`.
3. Run the test suite with `pytest`.
4. Ensure code style with `ruff check src tests` and `black src tests`.

## Pull request process

- Provide a clear description of your changes.
- Update documentation and add tests where appropriate.
- Ensure CI checks pass before requesting a review.

## Reporting issues

Please file issues via GitHub with a minimal reproducible example. Include Python version, platform,
and any relevant stack traces.
